# 嵌入式GUI
### 1. 基本概念
1. 简介
   * Graphical User Interface,图形用户界面
   * 强调人-机-环境作为一个系统进行设计
2. 特点
   * 轻型/占用资源少/高性能/便于移植/可配源
### 2. 分类
1. 主流嵌入式GUI
   * Qt/E：
   * MiniGUI
   * MicroWindows
   * Tiny-X
   * Palm
   * WinCE
   * Android
   * iOS
   * 鸿蒙
### 3. 体系结构-分层设计
1. 嵌入式GUI
2. 抽象层
3. 核心层
4. 应用程序接口层
5. 种一层调用下一层的接口实现对本层功能的封装
* 初始化：初始化是指在一个系统或组件首次启动时执行的过程，以确保其处于已知且稳定的状态。在分层体系结构中，初始化通常按照层次自底向上进行
* 打开接口：在分层体系结构中，为上层提供访问下层服务的入口点。这个过程可以在初始化过程中完成，也可以在运行时动态进行
